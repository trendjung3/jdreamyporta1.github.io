WEB PUBLISHER
웹 제작을 즐기는 웹 퍼블리셔 강정임입니다.

웹 퍼블리셔이자 웹 디자이너로서의 안목을 높이기 위해 항상 공부하는 자세로 수 많은 웹 사이트를 접하고 코드를 분석하며 제 것으로 만드는 것을 즐깁니다. 웹 제작을 하는 데에 있어 디자인의 질을 높이는데 최선을 다할 것이며 사용자들을 고려해 유저 인터페이스에 많은 노력을 다하겠습니다.또한 사용자와의 원활한 커뮤니케이션을 통해 즐기며 일하겠습니다. 그리고 현재에 안주하는 디자이너가 아닌 끊임없는 자기 계발로 웹 제작에서 최고의 결과를 낼 수 있도록 하겠습니다. 좋은 웹을 만들기 위해 항상 제자리에 있지 않고 발전해나가는 웹 퍼블리셔가 되기 위해 노력하겠습니다.
